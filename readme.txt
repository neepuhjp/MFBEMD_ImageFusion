
***********************************************************************************************************************************************************
This package contains the MATLAB code which is associated with the following paper:

"Novel and Fast EMD-based Image Fusion via Morphological Filter, The Visual Computer, https://link.springer.com/article/10.1007/s00371-022-02588-x" 

Authors: Qi Xie, Jianping Hu, Xiaochao Wang, Daochang Zhang,  Hong Qin .

Usage of this code is only free for research purposes. Please refer to the above publication if you use this code. 

Code edited by Jianping Hu, email: neduhjp307@163.com,  Northeast Electric Power University.

********************************************************************************************************************************
Usage:

There are two demo files:

(1) batchImgPatchFusionMain.m is used to test the proposed method for many pairs of source images;

(2) sigImgPatchFusionMain.m is used to test the proposed method for a pair of source images;
